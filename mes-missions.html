Optimisation du référencement (SEO)

Lors de mon stage, j’ai également été chargée d’optimiser le référencement du site internet. Pour ce faire, j’ai utilisé une extension Google qu’un des alternants de l’entreprise m’avait conseillée : Lighthouse SEO. Cette extension permet de répertorier tous les problèmes impactant le référencement, l’accessibilité mais aussi les performances. Voici à quoi ressemble un diagnostic de l’extension sur la page d’accueil :

L’extension permet également de voir les performances du site au moment du diagnostic et propose également des solutions pour améliorer le score du site internet en fonction des catégories :

Pour améliorer le référencement, j’ai donc appliqué les recommandations de l’extension, et plus particulièrement au niveau des formulaires. En effet, ces derniers étaient dénués de labels. Pour des raisons d’accessibilité, j’ai dû les ajouter dans les différents composants des formulaires du site internet. Tout d’abord, je vais vous expliquer grâce à des extraits de code comment fonctionne un composant et comment lui transmettre des variables afin d’en changer le comportement.
Prenons pour exemple le formulaire de connexion. Ce dernier est composé de trois composants : deux « FieldInput » servant à entrer du texte brut et un bouton de validation. Nous nous intéresserons donc aux composants « FieldInput » auxquels nous devons transmettre un label. Le composant « FieldInput » est constitué d’une balise « div » entourant une balise « input ». Les caractéristiques utilisés dans la balise « div » sont des classes CSS prédéfinies. La balise div contient donc le visuel de l’input. La balise « input » quant à elle contient le comportement de l’input au sein d’une page (événements, taille maximum, type…).

Ce composant contient toutes les possibilités de style et de comportement utilisé sur le site internet. Lorsque ce composant doit apparaitre sur une des pages du site, il faut qu’on l’importe dans la partie script comme ceci :

Nous avons importé deux composants pour créer notre formulaire de connexion. Nous voyons donc que parmi les caractéristiques du composant, le label a été ajouté. J’ai donc effectué cette opération sur tous les composants utilisés pour les formulaires.
